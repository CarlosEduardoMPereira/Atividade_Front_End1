SOBRE CONHECIMENTOS OBJETIVO

CONHECIMENTOS:

Meu primeiro contato com a programação foi ao criar o projeto Codaí, um Sistema de Controle Financeiro. Desenvolvi funcionalidades para gerenciar entradas, saídas e histórico de despesas, juntamente com páginas para login e logoff. A aplicação foi desenvolvida de forma responsiva para garantir uma boa experiência em diferentes dispositivos.

Até o momento, adquiri uma compreensão básica em programação, abrangendo as seguintes tecnologias:

  1. Github
  2. HTML
  3. CSS
  4. JavaScript
  5. Bootstrap

Durante o desenvolvimento do projeto Codaío, utilizei o GitHub para o versionamento do código, colaboração com outros desenvolvedores e hospedagem do projeto. Em relação às linguagens e frameworks, empreguei HTML para estruturar o conteúdo da aplicação, CSS para estilização e design, JavaScript para funcionalidades interativas e dinâmicas, e Bootstrap para criar um layout responsivo e moderno de maneira mais rápida e eficiente.
